## 📌 Project Overview

This is a **SAP Fiori-style Material Stock Report** web application 
that simulates the behavior of an **ABAP ALV (ABAP List Viewer) 
Grid Report** — one of the most commonly built reports in real 
SAP enterprise environments.

Built using **HTML5, CSS3, JavaScript and SAP OpenUI5 design 
principles**, this app demonstrates how SAP ABAP developers 
create data reporting interfaces in modern Fiori applications.

## ✨ Features

| Feature | Description |
|---|---|
| 🔍 Multi-field Filter | Filter by Material ID, Description, Plant, Category, Status |
| 📊 Summary Tiles | Live count of Available / Low Stock / Out of Stock |
| 🚦 Status Indicators | Color-coded badges (Green / Orange / Red) |
| ⬇️ CSV Export | Download filtered data as CSV file |
| 📱 Responsive | Works on desktop, tablet and mobile |
| ⚡ Zero Dependencies | No CDN needed, works fully offline |
